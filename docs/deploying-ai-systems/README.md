# Deploying AI Systems

Tài liệu này sẽ là track dành cho việc triển khai và vận hành hệ thống AI trong môi trường thực tế.

Các hướng nội dung dự kiến:

- kiến trúc triển khai model serving
- batching, caching, autoscaling
- deployment với container và Kubernetes
- vector database và retrieval services trong production
- observability, alerting và incident handling
- cost, latency và capacity planning
